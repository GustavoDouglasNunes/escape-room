## 🎯 Gameplay

### Sala 1 - O Quarto Trancado

- Quadro → **7**
- Cama → **4**
- Livro → **2**
- **Código final: 742**

### Sala 2 - Biblioteca

- Livro 1 → F
- Livro 2 → U
- Livro 4 → G
- Livro 5 → IR
- **Palavra final: FUGIR**

### Sala 3 - Saída Final

- Ordem correta: **SOL → LUA → ESTRELA**
